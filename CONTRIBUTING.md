# Contribuer à GestSIS_Doc

## Environnement de développement

Ce dépôt contient la documentation officielle de GestSIS, générée avec [Retype](https://retype.com/). Pour prévisualiser la documentation localement avec tous les autres services, utilise le dépôt [GestSIS_Dev_docker](https://github.com/GestSIS/GestSIS_Dev_docker).

La documentation est accessible sur http://localhost:8081 une fois les services démarrés.

## Structure

Les fichiers sources sont dans le dossier `src/` au format Markdown. Retype gère la génération automatiquement à chaud — toute modification est visible immédiatement dans le navigateur.

## Contribution

Pour le workflow de contribution, les conventions et le signalement de bugs, voir le [guide de contribution principal](https://github.com/GestSIS/GestSIS_Dev_docker/blob/main/CONTRIBUTING.md).
